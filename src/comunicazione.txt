Standard per la comunicazione: csv

Aggiungi player
    Il client invia: ADD;NOMEPLAYER;IDPEDINA
    Il server risponde: 0;TESTOERRORE = errore
                        ADD;
Avvia partita
    Il client invia: START;
    Il server risponde: 0;TESTOERRORE = errore
                    	START;{INFOPLAYER1},{INFOPLAYER2}
Lancia i dadi
    Il client invia: ROLL;DADO1;DADO2
    Il server risponde: 0;TESTOERRORE = errore
                        ROLL-0;{INFOPLAYERAGGIORNATE}			(quando capita su una proprieta)
                        ROLL-1;{INFOPLAYERAGGIORNATE};TESTO	(quando capita sulle altre caselle)
Acquisto casella
    Il client invia: BUY;
    Il server risponde: 0;TESTOERRORE = errore
                        BUY;{INFOPLAYERAGGIORNATE}
Termina turno
    Il client invia: CH;
    Il server risponde: 0;TESTOERRORE = errore
                        CH;IDPLAYERSUCCESSIVO
Ipoteca casella
    Il client invia: IP;IDCASELLA
    Il server risponde: 0;TESTOERRORE = errore
                        IP;{INFOPLAYERAGGIORNATE}

Lista dei nomi dei player e la relativa pedina
    Il client invia: LST;
    Il server risponde: LST;NOMEPLAYER1-IDPEDINA1,NOMEPLAYER2-IDPEDINA2...

Informazione della casella
    Il client invia: INFC;IDCASELLA
    Il server risponde: INFC;{INFOCASELLA}

Fine del gioco
    In risposta al change turno
    Il server invia: END;{INFOPLAYER}
